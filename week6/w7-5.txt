Procedure
1. User Login​​

    ​​Step 1​​: User enters username and password on the ​​Login Page​​.
    ​​Step 2​​: System validates credentials.
        ​​If valid​​: Redirects to ​​Enrollment Page​​.
        ​​If invalid​​: Returns error message, prompts re-entry.
​​2. Course Enrollment​​
    ​​Step 3​​: On the ​​Enrollment Page​​, user selects desired courses.
    ​​Step 4​​: System checks:
        Course availability (e.g., open slots, prerequisites).
        Conflicts (e.g., schedule overlaps).
    ​​Step 5​​: System responds:
        ​​If valid​​: Confirms enrollment, updates records.
        ​​If invalid​​: Returns error (e.g., "Course full" or "Schedule conflict").
